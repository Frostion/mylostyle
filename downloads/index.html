firmware updates

Make sure you select the update file that matches your model (US or JP) of Mylo! Firmware update tutorial

Firmware update v1.200 (for US model)
Adds new functionality to the Mylo COM-2, including the ability to record videos with the built-in camera, a new "Games" section in the main menu, and support for 16GB Memory Stick Duo cards. Source: rebyte.me

Firmware update v1.100 (for JP model)
Adds new functionality to the Mylo COM-2, including the ability to record videos with the built-in camera, a new "Games" section in the main menu, and support for connecting to Wi-Fi networks via Buffalo AOSS. Source: Sony Japan's website
