how to install firmware updates

In this tutorial, you'll learn how to install a firmware update onto your Mylo.

  1. Download the update.bin file for your model of Mylo. If you have a US model, download the US firmware update, and if you have a Japan model, download the JP firmware update instead.
  2. Connect your Mylo to AC power and plug it into your computer via USB. The Mylo's display should show "MSC (Mass Storage Class) mode" and it should show up as a drive on your computer.
  3. Copy the update.bin file to the root (top level) of your Mylo's internal storage (or Memory Stick if you have one).
  4. Disconnect the Mylo from your computer, but leave it connected to AC power.
  5. On the home screen of your Mylo, select "Tools", then select "System Update".
  6. A dialog box will pop up asking you for the location of the update file. Select whether you copied the update.bin file to the Mylo's interal storage or to the Memory Stick, then click "OK". (If you don't have a Memory Stick inserted, then the internal storage will be the only option available on the dialog box.)
  7. Another dialog box will pop up, asking you to confirm if you want to install the update. Click "OK" to continue.
  8. The Mylo will shut down, then display "Preparing to update", then "Updating". The update took about 2 minutes to install on my Mylo.
  9. After the update is complete, the Mylo will display "System update is complete". Touch the screen to continue.
  10. The Mylo will reboot and your update is now installed! You may now delete the update.bin file on your Mylo to free up space.
